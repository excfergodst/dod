Detailed Description

Base Event Interface class.

RishEngine Events.

Note
Always dispatch your events with rl::EventDispatcher in your own layer which is inherited from rl::Layer in the overrided function void onEvent(rl::Event &event)
Warning
Use rl::EventDispatcher instead of using if(event.getEventType() == rl::EventType::XXX) to check the event type
